The Mathematics

The round robin is a one-factorization of Kn

Model the field as the complete graph Kn: one vertex per player, one edge for every required match. A legal round is a set of matches in which nobody appears twice. In graph terms that is a perfect matching, also called a one-factor: no player is double booked. A schedule that plays every edge exactly once therefore partitions the edge set of Kn into rounds, which is a one-factorization. The circle method constructs one directly: fix one seat, rotate everyone else one seat per round, and pair opposite seats.

Elo expectation and update

E_A = 1 / (1 + 10^((R_B - R_A) / 400))
R_new = R_old + K (S - E),  K = 32,  S = 1 win, 0 loss

Every simulated match draws its winner from exactly these odds. The race score model then finds the per rack probability whose race outcome matches E, so favorites usually win, upsets stay possible, and scorelines look like real pool.

One round, one perfect matching

The seat with the ring never moves; everyone else rotates one seat per round. Opposite seats play, so each round draws parallel chords: a perfect matching. Faint lines are all the edges of Kn; stepping through the rounds, the bold chords cover each edge exactly once. A dashed chord marks the bye seat when the field is odd.

Seeding the knockout

A known limitation, acknowledged

If B meets A immediately after A faced C, B is said to receive a carry-over effect from C, for example facing an opponent right after a bruising loss. A schedule is considered balanced when these effects spread evenly. The circle method does the opposite: it attains the maximum possible carry-over effect value, and every schedule with maximum carry-over arises from the circle method. For a student union tournament this is an acceptable trade for simplicity, but fairer constructions exist, such as Russell's designs, balanced whenever n is a power of two.

K. G. Russell, Balancing carry-over effects in round robin tournaments, Biometrika 67 (1980) 127-131.
E. Lambrechts, A. M. C. Ficker, D. R. Goossens, F. C. R. Spieksma, Round-robin tournaments generated by the circle method have maximum carry-over, Mathematical Programming 172 (2018) 277-302.
